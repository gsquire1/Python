los – Loss of Sync lol – Link Failure losig – Loss of Signal swtxto – switch tx timeout swrxto – switch rx timeout proto – protocal error crc – crc with g_eof lrout – link reset out lrin – link reset in encout – encoding out error encin – encoding in error c3to – c3 timeout discard pcserr – pcs error ( ITW errors ) statechange – statechange txcrd_zero – Port TX credit zero
